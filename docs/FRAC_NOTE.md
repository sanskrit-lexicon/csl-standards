# FrAC Note

FrAC means Frequency, Attestation and Corpus information. It is an OntoLex
extension for connecting lexical data to corpus evidence such as attestations,
frequency counts, and corpus metadata.

Official reference: https://ontolex.github.io/frequency-attestation-corpus-information/

For these repositories, FrAC is not a dictionary-atlas feature. It needs real
corpus evidence, so it should wait for VisualDCS or another corpus source.

Allowed now:

- document where FrAC would fit later;
- note which OntoLex pilot cases would need attestation or frequency evidence;
- preserve the migrated FrAC-style pilot nodes as explicitly legacy stress-test
  material.

Not allowed now:

- invent FrAC evidence from dictionary-only records;
- expand the migrated FrAC-style pilot as if it were corpus evidence;
- build FrAC dashboards inside `csl-atlas`;
- mix FrAC with `csl-observatory` unless measuring repository activity around
  FrAC work.
